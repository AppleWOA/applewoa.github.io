Known issues to be aware of

Non functional peripherals

Cameras on any device currently is not functional and cannot be gotten to work

Operating System related issues

Due to limitations on the Windows Client Operating System, Dual SIM functionality is not working.

On dual sim devices, please avoid booting with a SIM inserted in the second slot, this may cause issues that may prevent you from using Cellular even after removing the SIM.

Call functionality is not currently automatically registered on builds higher than 18363 (1909). It can however be manually registered to work on higher builds.

Going to cellular advanced options on build 18363 (1909) and lower may crash the settings app. Please refer to Windows Configuration Designer documentation in order to externally provision an APN if needed.

SMS/Texts are not functional on builds higher than 18363 (1909).

MMS is not functional on any Windows version.

System instability and performance issues

A number of non original or counterfeit or third party replacement batteries are known to incorrectly report the correct capacity and charge state causing intermittent shutdowns, power cuts and overall system instability.

Thermals are known to not be on par with Windows 10 Mobile due to the Windows Client Operating System being more resource instensive than its Mobile counterpart.

Driver Update Channels

The drivers are available via two update channels:

The Experimental update channel contains work in progress, unstable/rough functionality. This update channel contains all the features you're used to from previous releases, but might have thermal issues, performance issues, or be very unstable.

The Stable update channel contains validated, stable, code. However this channel is missing many features you've come used to from past updates. Currently this update channel does not contain support for: GPU, Cellular, Wifi, Desktop bridges, Automatic Brightness. It is however stable, will not overheat as much as the Experimental channel, and fully usable.

Installations using WOA Deployer will default to the Stable channel. The Experimental channel is available using DriverUpdater and the provided Lumia-Drivers-Experimental.zip file on Github.
